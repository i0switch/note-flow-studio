# 11. 補足: 初期版の確定仕様まとめ

## 11-1. 初期版で確定している内容
- 買い切り配布
- ローカル動作
- UIはブラウザ
- 準自動
- 品質ルールはプロンプト管理
- note保存は非公式API + ブラウザ自動操作フォールバック
- browser automation は Playwright と PinchTab を併用可能
- noteアカウントごとに別設定
- 画像生成仕様はカスタム可能
- 参考資料の取込と再利用を行う
- オールジャンル対応の内部構造を持つ
- 記事連動アイキャッチとグラフを生成可能
- 無料部分 / 有料部分 / 導線文を保持する
- note販売設定の自動入力に対応する
- 入力項目は以下
  - キーワード
  - 対象ジャンル
  - 使用アカウント
  - プロンプト名
  - 参考資料
  - 画像あり/なし
  - グラフあり/なし
  - 販売モード
  - 補足指示
- 保存単位は履歴・生成物・保存結果を含めて採用
- 再編集あり
- 将来X対応を視野に入れる
