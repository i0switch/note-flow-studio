## 2-2. 画面仕様書

### SCR-001 ダッシュボード / 記事生成画面
**目的**
記事生成の起点となる主画面。

**主な入力項目**
- キーワード
- 対象ジャンル
- 使用アカウント
- プロンプト名
- 参考資料
- 画像あり/なし
- グラフあり/なし
- 販売モード（通常 / 無料+有料）
- 想定価格
- 補足指示
- 本文生成AI
- 画像生成方式

**主な表示項目**
- 現在の既定設定
- 参考資料の使用件数
- 直近実行結果
- 実行状態
- 保存先方式
- フォールバック経路
- 生成予定アセット（アイキャッチ / グラフ / CTA）

**主な操作**
- 記事生成開始
- 下書き保存まで実行
- 生成だけ実行
- 参考資料を選択して紐付け
- フォールバック設定確認

### SCR-002 実行履歴一覧
**主な表示項目**
- 実行日時
- キーワード
- ジャンル
- アカウント名
- プロンプト名
- 本文生成AI
- 販売モード
- 保存結果
- フォールバック有無
- 使用保存方式
- 下書きURL
- ステータス

### SCR-003 記事詳細 / 再編集画面
**主な表示項目**
- タイトル
- リード
- 無料部分
- 有料部分
- 無料→有料誘導文
- 本文
- 参考資料一覧
- 生成グラフ一覧
- アイキャッチ画像
- 画像プロンプト
- 生成画像一覧
- 販売設定プレビュー
- 保存ログ
- エラーログ
- 保存アダプタ履歴

**主な操作**
- タイトル再生成
- 本文再生成
- 誘導文再生成
- グラフ再生成
- 画像再生成
- noteへ再保存
- note販売設定を自動反映
- 他アカウントへ保存
- 保存方式を指定して再保存

### SCR-005 プロンプト編集画面
**入力項目**
- プロンプト名
- 用途
- 対象媒体
- 対応ジャンル
- 本文システムプロンプト
- 本文ユーザープロンプトテンプレート
- 参考資料取り込みテンプレート
- 無料→有料導線テンプレート
- グラフ生成テンプレート
- 画像プロンプトテンプレート
- 推奨AI
- 既定文字数帯
- メモ

### SCR-007 noteアカウント編集画面
**入力項目**
- 表示名
- 保存方式優先順位
- ブラウザ操作方式優先順位
- Playwright セッション設定
- PinchTab プロファイル設定
- 既定プロンプト
- 既定画像設定
- 既定販売プロファイル
- フォールバック設定

**対象**
- Gemini API
- OpenAI API
- Claude API
- Codex OAuth

### SCR-009 画像生成設定画面
**入力項目**
- 有効/無効
- 生成方式
- 生成枚数
- 用途（アイキャッチ / 挿絵）
- 記事内容連動アイキャッチ有無
- 比率
- サイズ
- スタイル補足

### SCR-010 システム設定画面
**入力項目**
- localhost ポート
- ログ保存期間
- 自動バックアップ有無
- 再試行回数
- タイムアウト秒
- デバッグログ有無
- 参考資料保存先
- ジャンル自動判定有無
- 販売設定既定値
- PinchTab ベースURL
- PinchTab トークン使用有無

### SCR-011 接続確認 / 診断画面
**確認対象**
- 各AI API接続
- Codex OAuth状態
- note API疎通
- Playwright起動可否
- PinchTab疎通
- アカウント別保存テスト
- 販売設定入力テスト

### SCR-012 ブラウザ操作設定画面
**目的**
ブラウザ自動操作の詳細設定を管理する。

**入力項目**
- 既定ブラウザ操作方式
- Playwright 有効/無効
- PinchTab 有効/無効
- PinchTab ベースURL
- PinchTab 認証トークン
- Playwright 永続プロファイルパス
- PinchTab プロファイル名ルール
- 操作前確認有無

### SCR-013 参考資料管理画面
**目的**
記事生成に使う参考資料を取り込み、検索し、ジョブへ紐付ける。

**主な入力項目**
- 取込方法（URL / テキスト / ファイル）
- タイトル
- 参考元URL
- 要約
- タグ
- 関連ジャンル
- 信頼度メモ

**主な表示項目**
- 参考資料一覧
- 抽出テキスト
- 使用可能ステータス
- 紐付け済みジョブ数

**主な操作**
- 新規取込
- 再解析
- ジョブへ追加
- 無効化

### SCR-014 販売設定 / 有料導線画面
**目的**
無料部分から有料部分への導線、価格、販売設定を管理する。

**主な入力項目**
- 販売モード
- 価格
- 無料公開範囲
- 導入CTAテンプレート
- 有料部分導線テンプレート
- 特典説明
- ターゲット読者

**主な表示項目**
- 無料部分プレビュー
- 有料部分プレビュー
- 誘導文プレビュー
- note 販売設定適用状況

**主な操作**
- 誘導文生成
- 価格候補生成
- 販売設定を記事へ反映
- note へ販売設定込みで保存
